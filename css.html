Overview

To avoid name collision, all classes defined in UI Schema start with the prefix ui-. They are divided into modules for the ease of maintenance and optimization. Unlike Bootstrap, UI Schema is not declared to be mobile first. We have observed the fact that screen resolutions keep on growing both for smartphones and laptops.
