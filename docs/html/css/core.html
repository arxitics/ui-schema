Font sizing

We use the following media queries to set the value of rem unit:

min-width N/A 768px 992px 1248px 1334px 1408px 1568px 1888px 2016px
rem unit 8px 9px 10px 11px 12px 13px 14px 15px 16px

Then all font sizes are specified with rems and ems. For headings, we have

Heading h1 h2 h3 h4 h5 h6
font-size 3.6rem 3rem 2.4rem 1.8rem 1.4rem 1.2rem
